Where can I get help?

The places where you can get help as part of the Australian BioCommons effort.

Community Discussion Groups

Email discussion groups for various bioinformatics communities of practice:

  • Genome assembly
  • Genome annotation
  • Microbiome analyses
  • Proteomics
  • Metabolomics

Training material

  • Search the TESS training portal which contains material used in 1000s of bioinformatics training courses.
  • Check out Melbourne Bioinformatics extensive range of tutorials and protocols
  • Galaxy Training Network: contains over 100 tutorials that can be undertaken in your own time on the free to use Galaxy Australia service. Tutorial categories include Assembly, Epigenetics, Genome Annotation, Metabolomics, Metagenomics, Proteomics, Sequence Analysis, Transcriptomics and Visualisation.
  • Watch recordings from the BioCommons webinar series which includes a wide variety of content surrounding many bioinformatics tools and approaches.
  • Upcoming BioCommons Events.
  • For information on how to get started using High Performance Computing (HPC) environments, check out:
    • HPC-carpentry
    • Swiss Institute of Bioinformatics: High Performance Computing in Life Sciences
    • Melbourne Bioinformatics workshop: “High-Performance Computing”

Training workshops

  • Search the TeSS Training Events calendar for upcoming bioinformatics training events being held in Australia

Documentation

  • bio.tools: is a searchable registry of bioinformatics tools containing descriptions and links to instances, manuscripts etc.
  • BioContainers: is a searchable registry and repository of bioinformatics software that has been containerised (using docker, bioconda and singularity) to enable easy installation on various computational systems. Check out the recordings of BioCommons webinars and training workshops about containerisation.
  • WorkflowHub [about / main hub page]: workflow registration and deployment
  • BioCommons GitHub: this is the default location for tool and workflow documentation, and includes:
    • useful links (including to source developement repositories and sites),
    • tutorials and install instructions,
    • compute resource usage summaries, and
    • information specific to deployment of these tools and workflows across BioCommons partner infrastructures
